# C&Cサーバー(Command & Controlle)
*マルウェア感染した端末を踏み台に遠隔操作(Command & Controle)等を行う際に指令塔となるサーバー*

- サーバー攻撃の原価
    - 従来
        - 無差別な攻撃
    - 2026年
        - 標的型の攻撃

マルウェアに感染している端末をBotとして、様々な攻撃の踏み台とする
- DDos攻撃
    - 特定のシステムへの過負荷攻撃
        - C&Cサーバーからクラウドサービスなどを経由する事であたかもHTTPS通信であるかのような振る舞いをする
            - マルウェアへ感染するとport番号80などを使用して、Web閲覧などが行われるため、発見する事すらむずかしくなる

## 攻撃手法分類

1. 標的型攻撃

- 特定の企業や個人を標的にして、攻撃を行う
    - 事前に対象などを決めた上で実行しているため、実行が始まった時には高速で攻撃が進行する事が予想される

2. 大規模攻撃

- 事前に構成されたボットネットに対して一斉に命令を送る事でサービスの停止、スパムメールなどを送信する攻撃
    - 利用されているボットネットは利用されている事に気づく事は出来ないため、気付かぬ内に利用されていたという可能性がある

3. 待ち受け型攻撃

- C&Cサーバーで、正規なWebサイトを装って不正な通信を行なって感染させる攻撃
