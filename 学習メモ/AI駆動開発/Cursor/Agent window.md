# Cursor3を2週間使って感じたAgent Windowの魅力

4/2 Cursor3のリリースによってAgent Windowが使いやすくなった

## 何が変化したか？

主役の変化
- Editor WindowからAgent Windowへ
    - Agentsとの会話ペインを中心に人間は最終調整というイメージ

- マルチワークスペース
    - ワークスペースが分離される事でコンテキストがワークスペースに閉じるかつ、複数のリポジトリに対するエージェント操作が格別にしやすくなった
        - 複数のエージェントを使っていてもクリック一回でエージェントを切り替える事ができるようになった。

- タブの機能が充実
    - エディター
        - LSPがしっかり効いている
    - ブラウザ
    - ターミナル

- Git操作もGood
    - Brachの状態に応じて適切なアクションを行えるコミットやPushなどのボタンが出現する

- ローカルとクラウドの行き来が自然になった
    - 今まではPCを閉じてしまうと作業が中断されてしまうなどという事が起こり得ていた
        - そこにクラウドに渡すという事ができるようになる事でより体験が向上
