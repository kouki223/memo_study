### 問題の原因

**3つの変更が問題だったとのこと**

原因：
1. デフォルトの推論負荷の変更
2. セッション情報の削除におけるバグ
3. システムプロンプトの変更

#### デフォルトの推論負荷の変更
Opus 4.6リリース時に推論負荷をhightに設定したとのこと
その影響で出力は高品質になった反面、UIがフリーズしたような状態になることがあり、さらに、そのUIとの差分により、トークン使用量などに不均衡が生じているように見えた。（ユーザーからのFBによるもの）
