# Amazon Bedrock

## 適切なタイミングでモデルとエージェントを選択してくれるサービス

生成 AI アプリケーションやエージェントの構築向けの包括的なマネージドプラットフォーム。実験（PoC）から本番デプロイまで一貫して利用できる。

### 提供するもの

- **基盤モデル（FM）** — 複数ベンダーのモデルを選択・利用
- **エージェント** — デプロイ・運用のためのサービス
- **カスタマイズ** — ファインチューニング、保護、最適化
- **ナレッジベース** — アプリケーションを最新データに接続

## 主な機能

| 機能 | 内容 |
|---|---|
| モデルの選択 | 数百のFMと評価ツールにアクセス。パフォーマンス・コストに応じて最適なモデルを選べる |
| データを利用してカスタマイズ | ファインチューニングなどで自社データに合わせた調整 |
| 安全とガードレール | コンテンツフィルタ・ポリシー制御 |
| コストの最適化 | 利用量・モデル選択によるコスト管理 |
| オーケストレーション | エージェント・ワークフローの統合運用 |

## AgentCore（エージェント本番運用）

PoCから本番への移行を支援するエージェント基盤。スケール・信頼性・セキュリティを前提に設計されている。

| コンポーネント | 役割 |
|---|---|
| AgentCore Gateway | ツール・データ・他エージェントの検出と安全な接続 |
| AgentCore Memory | 短期・長期メモリの保持 |
| AgentCore Runtime | 大規模・安全なエージェントデプロイ。非同期ワークロード向けの長時間セッション対応 |
| AgentCore Identity | エージェント認証 |
| AgentCore Observability | エージェントの動作監視・安全性の確保 |

## ユースケース例

- **コンテンツ生成** — ブログ、SNS、Webコピーなどの文章作成
- **バーチャルアシスタント** — リクエスト理解 → タスク分解 → マルチモーダル分析 → アクション実行
- **文書要約** — 記事・レポート・論文・技術文書・書籍の要約
- **画像生成** — 広告・Web・プレゼン用のビジュアル作成
- **ワークフロー自動化** — 定型タスクの自動化、財務報告・需要予測など部門横断の業務効率化
