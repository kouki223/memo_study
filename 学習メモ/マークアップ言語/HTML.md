# HTMLについて
**HTML => ハイパーテキストマークアップ言語**

## `<iframe>`
- インラインフレーム要素
	- HTML の要素で、入れ子になった閲覧コンテキストを表現し、現在の HTML ページに別のページを埋め込むことができます。
- 要素
	- 複数のグローバル要素を持つ
		- 理論的には多くのiframeを使用する事ができるが、パフォーマンスの問題を確認する事は重要

## `<ruby>` ルビ注釈要素

- 役割
	- ベーステキストの上や下、隣に描画される小さな注釈
		- 東アジアにおける文字の発音などを示す時に使われる

- 語源
	- ルビは、読みやすさを保ったままテキストを新聞用紙に印刷できる最小のサイズを表す

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```

```html
<ruby>
  漢 <rp>(</rp><rt>Kan</rt><rp>)</rp> 字 <rp>(</rp><rt>ji</rt><rp>)</rp>
</ruby>
```

```html
<ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>
```
